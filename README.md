# consumer-complaint-analysis-excel

## Project Overview
This is a personal data analysis project that analyzes consumer complaint data using Microsoft Excel. The project demonstrates end-to-end data analysis skills including data cleaning, advanced Excel formulas, pivot table analysis, and dashboard creation to generate meaningful business insights.

---

## Tools & Technologies
- Microsoft Excel (Advanced)
- Advanced Excel formulas:
  - IF, IFS
  - SUMIFS, COUNTIFS
  - VLOOKUP / XLOOKUP
  - INDEX & MATCH
  - TEXT and DATE functions
- Pivot Tables & Pivot Charts
- Interactive Excel Dashboard
- Data sorting and filtering

---

## Dataset & Project File
- **Consumer complaint analysis Project.xlsx**
  - Raw dataset
  - Cleaned and transformed data
  - Pivot table analysis
  - Interactive dashboard

---

## Key Tasks Performed
- Cleaned and standardized consumer complaint data
- Handled missing and inconsistent values
- Applied advanced Excel formulas for calculations and logic
- Built pivot tables to summarize complaint trends and categories
- Designed an interactive Excel dashboard for insight reporting
- Analyzed complaint patterns across products, regions, and time periods

---

## Key Insights
- Identified products with the highest number of complaints
- Analyzed complaint trends over time
- Highlighted common consumer issues across categories
- Provided data-driven insights to support decision-making
